  # Color Gradient
   A package for generate colors between two colors

  ### Installation

  ```sh
  npm install color-gradient --save
  ```

  ## Usage

  ```js
  var generate = require('color-gradient');

  var colors = generate("#fff", "#000", 6);
  ```

  ### Test


  ```sh
  npm test
  ```
